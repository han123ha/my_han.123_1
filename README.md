# 文本统一替换工具

- 一款可以统一替换文本相同内容的软件！
- 其功能是是为了，更方便的替换文本中相同的内容。
- 使用说明

| 步骤 | 操作说明                                                                 |
| ---- | ------------------------------------------------------------------------ |
| 第1步 | 点击绿旗进入文本开始菜单                                                 |
| 第2步 | 点击中间方框进入文本上传界面，上传文本                                   |
| 第3步 | 开始编辑“点击，中间的‘文本统一修改器按钮’进入”                           |
| 第4步 | 你可以开始编辑文本啦！                                                   |
| 第5步 | 编辑完文本后可以点绿色的“导出按钮”把编辑好的文本保存到本地               |
